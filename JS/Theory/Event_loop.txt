Event loop регулирует последовательность исполнения контекстов - стек.

Процесс распределения вызовов: 
1) Выполнения синхронного кода - код добавляется в стек и выполняется
2) Микро-задачи: event loop проверяет код на наличие микро тасков и добавляет их в очередь на выполнение. Идет выполнение ВСЕХ микро тасков в очереди 
	Примеры микро-задач: Promise.then()/catch()/finally(), process.nextTick(), queueMicrotask()
3) Макро-задачи: идет проверка очереди макро задач и выполнение их по ОДНОЙ
	Примеры макро-задач: setTimeout, setInterval, setImmediate()

process.nextTick() - функция, позволяющая запланировать выполнение колбека в начало цикла микрозадач. node js
setImmediate() - добавляет код в очередь после всех макрозадач. node js

queueMicrotask() - добавляет колбек в очередь микро-тасков в конец

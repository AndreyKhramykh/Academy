while //
Синтаксис:
while (condition) {
	code
}
Code выполняется, когда condition истинно. Каждый вход в тело цикла называется итерацией.

do...while // Если мы хотим, что бы код в теле цикла выполнился хотя бы один раз, можно использовать этот цикл.

for (начало; условие; шаг) {
	код
}
Алгоритм цикла:
Выполнить начало
→ (Если условие == true → Выполнить тело, Выполнить шаг)
→ (Если условие == true → Выполнить тело, Выполнить шаг)
→ (Если условие == true → Выполнить тело, Выполнить шаг)
→ ...

break - выход из цикла полностью
continue - пропустить определенную итерацию цикла

Метка - позволяет выйти из вложенных циклов, пример:

метка: for (...) {
	for (...) {
		if (something) break метка
	}
}

Перебор объекта:

in - оператор проверки объекта на наличие свойства.
Пример:
let user = {
	name: 'some',
	surname: 'some'
}
console.log( 'name' in user) // true
console.log( 'surname' in user) // true
console.log( 'age' in user) // false

Цикл for (let key in obj) {
	code // key, obj[key] и тд
}

Перебор массива: 

Цикл for (let item of array) {
	item = элемент массива
}
